== Description ==

Um plugin para analisar o servidor.
Ele te mostra quais as configurações do seu servidor de hospedagem e te mostra o desempenho.
Ideal para sites que estão travando, hospedagem compartilhada, servidores fracos, e-commerce, sistemas integrados e muitos outros.

== Installation ==

1. Upload `server-analitycs` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
